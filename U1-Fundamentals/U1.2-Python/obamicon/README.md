# Obamicon

To run on Mac:
`$ python3 obamicon.py`

To run on Windows:
`$ python obamicon.py`

## Contents

* `obamicon.py`
    * This program will create a new image called "recolored.jpg" in the current directory.
* `obamicon-ext.py`
    * This program includes additional image filters that students may choose to program.
